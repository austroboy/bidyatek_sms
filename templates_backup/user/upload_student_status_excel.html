Instructions:
  1. Download the template to see the correct format
  2. Required columns: "User ID" and "Status"
  3. Valid Status values: "Active" or "Deactive" (case-sensitive)
  4. Only students with matching User ID will be updated
  5. If status in Excel matches database, no change will be made
  6. When changing from Active to Deactive, students will be automatically logged out
